# Customer Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI.

The goal is to understand purchasing patterns, customer demographics, and product trends.

## Tools & Technologies

* Python (Pandas, NumPy)
* PostgreSQL
* Power BI
* Jupyter Notebook

## Dataset

The dataset contains customer shopping information including:

* Customer ID
* Age
* Gender
* Product Category
* Purchase Amount
* Payment Method
* Purchase Frequency
* Shipping Type

## Project Workflow

1. Data cleaning using Python
2. Feature engineering (Age groups and purchase frequency)
3. Loading cleaned data into PostgreSQL
4. SQL queries for analysis
5. Power BI dashboard visualization
